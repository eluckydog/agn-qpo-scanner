# AGN QPO: 模型理论与立项文档

## 核心假设

活动星系核（AGN）的光变曲线中可能存在准周期振荡（QPO），其物理来源尚未明确。目前候选的物理机制包括：

- **吸积盘颤动**：内区不稳定性驱动低频调制（~10⁻⁴ Hz）
- **热斑/旋进**：吸积盘热辐射区绕黑洞旋进
- **喷流进动**：相对论喷流的方向调制
- **双黑洞轨道**：超大质量双黑洞系统轨道周期调制

## 我们的方法优势

### 为什么用 Hilbert 相位？

传统方法（Lomb-Scargle 周期图）假设严格周期，对**频率漂移、间隙采样、非平稳性**敏感度低。

Hilbert 瞬时相位：
- 不预设周期，直接从数据提取 φ(t)
- 对频率漂移天然鲁棒
- 输出 phase coherence 作为 QPO 显著性指标
- 已在太阳周期上验证（U(1) 模型，R²=0.689）

### Pipeline

```
ZTF 光变曲线 → 去趋势 → Hilbert 变换 → φ(t)
                                  ↓

```
